Question 16

Correct
Mark 4.00 out of 4.00
Flag question

Question text

In HTTP version 1.1, what is the first field below the request line?
Select one:
a.

Connection.

b.

Any of these, HTTP is structured in plain-text.

c.

Accept-language.

d.

User-agent.

Feedback

Correct, unlike protocols like IP and TCP, HTTP does not structure its fields depending on their byte position in the packet. Rather, a HTTP packet is structured in plain-text with delimiters (sp, cr, and lf) separating header lines, field names, and values. 
The correct answer is: Any of these, HTTP is structured in plain-text.
Correct
Marks for this submission: 4.00/4.00.
